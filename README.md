# 台风巴威（BAVI 202609）浙江实时影响图

纯静态单页应用，打开 `index.html` 即可使用（无需构建、无需服务器，`file://` 也能跑）。

## 功能

- 实时路径、多机构预报（中国/台湾/日本/美国）、7/10/12 级风圈 —— 数据来自浙江省水利厅台风 API，页面每 5 分钟自动刷新
- 中央气象台实时预警（NMC API，按市聚合）
- 雷达拼图与 24 小时降雨分布图层
- 浙江 11 地市可点击边界，评估各市影响等级并给出防御建议
- 各市官方防御措施快照（`assets/measures.js`），滚动播报红/橙预警与最新措施
- 定位用户位置，优先展示"我的位置"周边情况
- 移动端适配：侧边抽屉、底部摘要条、市中心脉冲提示

## 文件结构

- `index.html` —— 全部页面与逻辑（单文件）
- `assets/zhejiang.js` —— 浙江 11 市边界 GeoJSON（内嵌）
- `assets/measures.js` —— 各市官方措施快照数据
- `tools/update-measures.ps1` + `update-prompt.txt` —— 本地定时任务脚本，用 `claude -p` 无头搜索最新官方通报并重写 measures.js（校验失败自动回滚）

## 调试参数

- `#loc=lat,lng` 模拟定位
- `#open=城市` 展开指定城市卡片
- `#pop=城市` 打开地图弹窗

注意：部署后 `assets/measures.js` 是静态快照，需重新推送才会更新；路径/预警/雷达/降雨等 API 图层在任何部署环境下均为实时。
